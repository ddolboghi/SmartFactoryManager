# Smart Factory Manager

- 디지털스마트부산 오전반 3조 SFM의 최종 프로젝트입니다.

# 주제

- 자동차 부품 제조 공정에서 AI를 활용한 양불판정과 예지보전

1. 도어 양품 판별
2. 윈드쉴드 사이드 몰딩 양품 판별
3. 열풍건조 설비 이상탐지
